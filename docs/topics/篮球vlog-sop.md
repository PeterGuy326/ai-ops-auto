# 篮球 vlog 运营 SOP

> 篮球 vlog 专题（NBA / CBA / 野球场）。
> 本文件是 篮球 vlog 专题的"业务事实底座"——给 LLM 当上下文，给 Topic Owner 当操作手册。
> 技术对抗（频率/指纹/养号）见 `docs/anti-risk.md`；平台改写风格见 `prompts/platform_style/*.md`。本文件不重复其内容。

## 1. 专题定位与边界

篮球 vlog 是面向"打球的 / 看球的 / 嘴炮的"年轻人专题，核心叙事是**"把篮球这件事拍得有节奏、嘴得有梗、看完想去打球"**。覆盖三层场景：

- **NBA**：当季话题（MVP 之争 / 季后赛 / 总决赛 / 重磅交易 / 球员动态），赛事节奏强、流量周期明确
- **CBA**：本土赛季 + 国家队大赛（亚洲杯 / 世预赛 / 奥运），叙事更偏"自己人"
- **野球场**：业余球员 vlog、训练 / 实战 / 装备 / 球场图鉴、最贴近用户日常

判定 checklist（满足全部即"对题"）：

- [ ] 主语是具体的"球员 / 球队 / 比赛 / 战术 / 装备 / 球场"——不是"我对篮球的看法"
- [ ] 必须带节奏感（视频 ≤ 90 秒优先 / 图文有强 hook），不能是"我今天打了个球"的流水账
- [ ] 调性年轻化：黑话、梗、节奏快、第一人称为主（"哥几个" / "兄弟" / "老铁" 均可，但避免低俗）
- [ ] 不说教 / 不写训练教程长文（那是另一个专题，本专题是 vlog 性质）
- [ ] 不涉及黑哨 / 黑球员家人 / 引战内容（红线见第 8 章）

跑题反例：
- ❌ "篮球运动的健康价值" —— 说教 + 没人看
- ❌ "三分球技术分解教学" —— 那是教学号，不是 vlog
- ❌ "我对中国男篮未来的看法" —— 没有具体场景 / 球员 / 比赛切入点

## 2. 受众画像

**主受众 A：18-30 岁年轻男性球迷 / 业余球员**

- 男性占比 ≥ 85%，二三四线城市比例高于一线
- 职业：在校生 / 互联网工作者 / 蓝领 / 自由职业；收入跨度大（学生党到年薪 40 万）
- 痛点：① 比赛太多刷不过来想要切片；② 自己打球录的视频没节奏；③ 想买装备但价格虚高；④ 嘴炮素材永远缺
- 高频搜索词："NBA 集锦"、"今日 NBA 战报"、"CBA 焦点战"、"球鞋测评"、"野球场扣篮"、"过人合集"、"球员评分"、"MVP 排名"
- 活跃时段：早高峰 7:00-9:00（看赛事结果）/ 午休 12:00-14:00 / 晚高峰 19:00-24:00（深夜赛事日 0:00-2:00 也有活跃）
- 平台偏好：抖音 = 视频号 > B 站 > 小红书 > 知乎 > 公众号 = 头条

**主受众 B：中学生 / 大学生球迷**

- 男性主导，预算敏感，"装备党"重灾区
- 高频搜索词："最便宜实战球鞋"、"野球场单挑"、"训练计划"、"AAU"、"高中篮球"
- 活跃时段：放学后 17:00-22:00 / 周末全天
- 平台偏好：抖音 > 视频号 > 小红书 > B 站 > 知乎

**主受众 C：30-40 岁怀旧球迷 / 老炮**

- 男性占绝对主导，看球年限 ≥ 10 年
- 内容偏好：经典比赛回放 / 球员退役回顾 / "我青春那年的篮球"
- 活跃时段：晚上 21:00-24:00 / 周末
- 平台偏好：B 站（深度向）> 知乎 > 公众号 > 视频号

| 平台 | 优先级 | 契合度（1-5） | 该专题策略 |
|---|---|---|---|
| 抖音 | 主攻（短视频） | 5 | NBA / CBA 集锦切片、野球场过人 hook、梗视频，30-60s 为主 |
| 视频号 | 主攻（短视频） | 5 | 同抖音内容平移，加"老炮怀旧"向；中老年球迷在视频号沉淀 |
| B 站 | 主攻（中长视频） | 5 | 3-10 分钟深度剪辑、球员生涯回顾、比赛战术分析、野球场长 vlog |
| 小红书 | 副本（高契合） | 4 | 球鞋测评、球场图鉴、训练装备、野球场打卡，女球迷市场也在涨 |
| 知乎 | 副本 | 3 | "如何评价 XX 球员"、"XX 战术为什么这样打" 问答 + 深度长文 |
| 头条 | 副本 | 3 | 战报 / 球员动态 / 怀旧文，触达中年球迷 |
| 公众号 | 副本 | 2 | 深度赛事复盘 / 球员长文；订阅黏性弱、转化低，不主推 |

**与 dws / 软考的最大差异**：篮球 vlog 是**短视频主导**的专题，抖音 / 视频号 / B 站三家是核心战场；公众号 / 知乎是支线、不是主战场。

## 5. 内容钩子库

### 开头钩子（10+ 条）

1. "兄弟们，今天这球我重看了 5 遍，必须发出来。"
2. "上来就第 3 球——这是 ESPN 隔天放 TOP1 的回合，预警。"
3. "野球场被打懵之后，我才知道这三件事我练错了。"
4. "刚下班开车 1 小时去打了个球，结果遇到了职业退役的。"
5. "这鞋我穿了 100 小时，今天必须把缺点全讲了。"
6. "5x5 数据是什么概念？整个 NBA 历史上不到 30 个人做到过。"
7. "杭州野球场鄙视链，我打了 12 个场子总结了一份名单。"
8. "你猜这场 CBA 总决赛吹了几次哨？答案震惊我自己。"
9. "30 秒看懂'电梯门'怎么跑，跑不会算我输。"
10. "退役一年的他，给我连续晃过 3 次——我录下来了。"
11. "兄弟，我今天要扯个梗，但你听完会笑死。"
12. "上来这一球——别滑走，第 8 秒有名场面。"

### 中段钩子（5+ 条）

1. "讲讲细节，这一步是分胜负的地方。"
2. "下面这段慢放，看他脚步——"
3. "等等——你听到刚才那声了么？回放给你听。"
4. "数据上看，他这一节投了 8 中 7，没记错的话。"
5. "OK 这球别走，名场面要来了。"
6. "兄弟们，这里我必须 cue 一下评论区，上次那个老哥说对了。"
7. "插一句吐槽——我真的不能理解这换防。"

### 结尾 CTA（5+ 条）

1. "兄弟们觉得这球该不该 TOP1？评论区报队。"
2. "下期讲 XX 战术，关注一下不掉车。"
3. "野球场图鉴系列还有 10 集，点头像追更。"
4. "球鞋包邮抽奖在主页置顶，关注 + 评论 + 转发参与。"
5. "你们野球场最强的人是谁？评论区甩名字。"
6. "三连一下，让算法把这条推给更多打球人。"

## 7. 发布节奏 SOP

**频率上限**：严格遵循 `docs/anti-risk.md` 全局规则——单账号每日 ≤ 2 条、最小间隔 4 小时、养号期 7-14 天内不发布、凌晨 0-6 点禁发。本节只写篮球 vlog 专题的差异化节奏。

**篮球 vlog 专题最佳时段**（赛事时间强相关）：

| 平台 | 工作日窗口 | 周末窗口 | 备注 |
|---|---|---|---|
| 抖音 | 7:00-9:00 / 12:00-14:00 / 19:00-23:00 | 9:00-12:00 / 14:00-23:00 | NBA 比赛日早 7-9 点是"看战报"高峰 |
| 视频号 | 7:00-9:00 / 12:00-14:00 / 19:00-23:00 | 同左 | 老炮 / 中年球迷在 21:00-23:00 集中 |
| B 站 | 19:00-23:00 | 14:00-17:00 / 19:00-23:00 | 周二 / 周四 21:00 是长视频黄金窗口 |
| 小红书 | 12:30-14:00 / 19:00-22:00 | 同左 | 球鞋 / 装备类周末上午也强 |
| 知乎 | 12:30-13:30 / 21:00-24:00 | 9:00-12:00 / 21:00-24:00 | 重磅赛事 / 重磅交易后 24h 是问答高峰 |
| 头条 | 7:00-9:00 / 17:30-19:30 | 9:00-12:00 / 17:00-20:00 | 中年球迷"看战报"早高峰是关键 |

**赛事节奏对齐**（这是篮球 vlog 与 dws / 软考的最大差异）：

| 赛事节点 | 内容策略 | 发布密度 |
|---|---|---|
| NBA 常规赛日（北京时间早 8:00-12:00 出比分） | 战报切片 / 名场面 / 数据梗 | 早 8:30-9:30 强冲一波 |
| NBA 比赛中（直播中） | 不建议发布；直播流自带流量，发了被淹没 | 暂停 |
| NBA 比赛日 12:00 后 | 完整集锦 / 单球员混剪 / 嘴炮复盘 | 12:30-14:00 + 21:00-23:00 |
| CBA 比赛日（19:35 / 20:00 开打） | 比赛后 22:30-23:30 战报 + 集锦 | 当晚 + 次日早 |
| 总决赛 / 全明星 / MVP 公布 | 流量峰值，提前 12h 预热稿 + 实时反应稿 + 24h 内深度复盘稿 | 三波连发 |
| 休赛期（每年 6-9 月） | 转会消息 / 球员训练 vlog / 怀旧向 / 野球场内容拉满 | 每周 +30% 野球场 |
| 季前赛 | 新赛季预测 / 球员评分 / 球队展望 | 每周常规 |

**每周条数建议**：

- 抖音 / 视频号：每周 5-7 条（短视频，密度高），赛事日可冲到日均 2 条但严守 4h 间隔
- B 站：每周 1-2 条长视频 + 1-2 条短视频
- 小红书：每周 3-5 条
- 知乎：每周 1-2 条（重大赛事或交易后必出一条）
- 头条 / 公众号：每周 1-2 条

**节假日**：
- 春节：NBA 不停赛，CBA 春节赛是流量小高峰
- 6-9 月休赛期：流量降 30-50%，必须转野球场 + 转会消息撑流量
- 大赛年（奥运 / 男篮亚洲杯 / 世界杯）：6-9 月反而是流量高峰，要提前布稿

**时间打散**：必须 `+ random(0, 600)s`，禁止整点发布——见 `docs/anti-risk.md` 第四层"行为层"。**特别提醒**：赛事结束后大批球迷号同时发战报，整点发布更容易被算法误判为"批量号"，时间打散是 vlog 专题的刚需。

## 8. 合规与红线

**通用红线**：
- ❌ 不黑哨 / 不带节奏"球员被收买" / 不引战球队球迷
- ❌ 不发布"赌球 / 滚球 / 让分"任何相关内容——抖音 / 视频号一查到位的违规
- ❌ 不黑球员家人（妻子 / 子女 / 父母），即便是梗
- ❌ 不引用未授权的 NBA / CBA 高清原片（短切片 < 30 秒 + 二次剪辑 + 评论字幕通常被算"合理使用"，但完整比赛回放发出去就是 100% 下架）
- ❌ 不"代购 / 卖鞋"未走合规渠道（球鞋灰色市场链路）

**版权红线（vlog 专题最高敏感）**：
- NBA 完整比赛 / 全场集锦 -> 必下架 + 警告
- CBA 完整比赛 -> 同上
- 短切片（≤ 30s）+ 自己解说 / 字幕 / 表情包 -> 一般通行
- 球员官方写真 / 海报 / 卡牌 -> 标注来源，慎用商用
- 自己拍的野球场 vlog -> 安全，但出现可识别的他人面孔需打码

**禁用词**：
- "黑哨" / "假球" / "收买" / "操纵"
- "赌一把" / "押 XX 赢" / "投注" / "滚球"
- "约 X" / "约球"以外的暗示性表达
- 黑话不能升级到低俗（"傻 X" / "废物 X" 这种避免）

**篮球 vlog 专属红线**：
- 引战球队球迷（"湖人粉都是 XX" / "勇士粉都是 XX"）
- 黑球员私生活
- 涉及 KD / 詹姆斯 / 易建联等头部球员的事实性指控未带来源
- 国家队相关内容必须正向，不带"恨铁不成钢"过线措辞

## 9. KPI 与异常处置

**健康基线**（篮球 vlog 专题，短视频量级远高于其他两专题）：

| 平台 | 单条 1h 数据 | 24h 数据 | 核心指标 |
|---|---|---|---|
| 抖音 | 播放 ≥ 3000 | 完播率 ≥ 40% | 评论 / 播放 ≥ 1.5%（嘴炮内容评论率是命脉） |
| 视频号 | 播放 ≥ 2000 | 完播率 ≥ 35% | 点赞 / 播放 ≥ 3% |
| B 站 | 播放 ≥ 1500 | 完播率 ≥ 30%（长视频） | 三连率 ≥ 5% |
| 小红书 | 曝光 ≥ 1500 | 完播率 ≥ 35% | 收藏 + 点赞 ≥ 6%（球鞋 / 球场图鉴收藏高） |
| 知乎 | 阅读 ≥ 1000 | —— | 赞同 ≥ 30 |
| 头条 | 阅读 ≥ 2000 | 完播率 ≥ 25% | 评论 ≥ 20 |

**篮球 vlog 的核心 KPI 是"评论率"**——嘴炮内容靠评论拉算法权重，单条 200 评论的破圈效果 > 单条 1 万播放无评论的纯曝光。

**限流触发阈值**：

| 信号 | 阈值 | 处置 |
|---|---|---|
| 抖音 24h 播放 < 中位数 20% | 单条 | 标记 DEGRADED；账号暂停 48h；下一条改"球员怀旧"等低敏选题 |
| 抖音"该作品涉及版权"提示 | 立即 | 下架；检查是否原片片段超 30s 或无二创 |
| 抖音 / 视频号"赌博 / 不良引导"提示 | 立即 | 下架 + 复盘文案是否带"押 / 赔率 / 输赢"敏感词；账号暂停 7 天 |
| B 站"内容审核中" > 24h | 单条 | 检查版权 / 引战 / 字幕风险；删去再投 |
| 评论区出现 ≥ 5 条引战 / 谩骂 | 单条 | 立即排查文案是否暗示性引战；触发后必须开评论精选模式 |
| 球员 / 球队公关下场（律师函 / 私信） | 立即 | 下架 + 公开致歉 + 复盘流程；不再发该球员负面 |

**异常处置 SOP**：
1. 触发后立刻 `update_health(DEGRADED)`（参考 `docs/anti-risk.md` 链路），暂停 48h
2. 复盘三件事：版权 / 引战 / 赌博词三红线是否触线
3. 复盘记录写到篮球 vlog 知识库 `events/yyyy-mm-dd-account-degraded.md`
4. 48h 后恢复，第一条必须是"野球场自拍 vlog"或"经典名场面怀旧"等纯安全选题
5. 重大赛事（总决赛 / 全明星）前 7 天必须扫历史内容，把"押 / 赌 / 黑哨"等灰色词洗一遍

## 10. 好坏样本对比

### 好样本 A（抖音 60s 切片）

> **标题**：杨瀚森这一球内线一打三，CBA 菜鸟里没人做得到
>
> **第 1 秒**：黑屏字幕"上来就第 3 球——这是今天 CBA TOP1"
> **第 3-30 秒**：精剪杨瀚森的内线连续动作，配合慢镜头 + 解说字幕（拆解脚步 + 节奏 + 终结）
> **第 30-50 秒**：数据截图（这一场 24+14+5+3 帽）+ 历史菜鸟同期对比
> **第 50-60 秒**：作者出镜或字幕："兄弟们，这评分该不该上 9？评论区报队。"

**为什么好**：① 强 hook 开头（直接告诉读者"第 3 秒看精华"）；② 数据 + 慢镜双轨；③ CTA 是"评论区报队"，直接引爆评论率（核心 KPI）；④ 60 秒长度匹配抖音算法；⑤ 完全合规（自己解说 + 二创字幕 = 合理使用）。

### 好样本 B（B 站长视频）

> **标题**：CBA 总决赛 G3 5 大判罚争议，逐帧告诉你哪几个吹错了
>
> **开头**：兄弟们，这场总决赛 G3 我看了 3 遍，5 个关键判罚我用慢动作逐帧拆。先说结论：3 个吹错 1 个有争议 1 个正确，今天逐帧打分。

**为什么好**：① 选题精准（具体赛事 G3）；② "5 个判罚"是结构化承诺；③ 开头给结论留扣子（具体哪 3 个吹错？只能看下去）；④ B 站长视频用户吃这套深度向；⑤ 完全不踩"黑哨"红线（用"吹错"而不是"黑哨"，是合规用语）。

### 坏样本 A（版权红线）

> **标题**：今日 NBA 湖人 vs 凯尔特人 全场集锦（高清完整版）
>
> **问题**：① 标题就触发版权检测（"全场"+"高清完整版"是 NBA 版权方扫描关键词）；② 即便侥幸上线 1 小时，被举报后下架 + 警告；③ 抖音 / 视频号会标记账号"重复版权违规"，第三次直接封号；④ 完全是无意义流量赌博，对账号长期价值是负资产。

### 坏样本 B（引战翻车）

> **标题**：勇士粉真的看不懂球，刚才这球还在嘴库里
>
> **问题**：① 标题直接引战（地域 / 球迷标签 + 贬低）；② 评论区会被勇士粉攻陷，触发举报阈值；③ 平台算法识别"球迷引战"会降权 7-14 天；④ 即使爆了流量，账号画像被算法标为"引战号"，后续优质内容也走不动；⑤ 对真球迷反作用——会被识别为低质嘴炮号。

### 坏样本 C（赌博词触线）

> **标题**：今晚湖人 vs 掘金，押湖人赢，赔率 1:1.8
>
> **问题**：① "押" + "赔率"是抖音 / 视频号违法宣传赌博的硬词；② 直接下架 + 账号封禁 7-30 天起步；③ 严重的会拉黑设备指纹，影响小号矩阵；④ 即便是开玩笑也不能用——平台审核没有上下文识别能力，词命中就违规。

### 坏样本 D（流水账）

> **标题**：今天去打球了
>
> **问题**：① 无 hook 无信息密度；② 算法判定"低质内容"权重稀释；③ 对 LLM 生成下一篇毫无参考价值（事实底座空）；④ 篮球 vlog 专题不允许这种"日记式" vlog，必须有具体球场 / 对手 / 名场面切入点。
